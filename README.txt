Установка скрипта блога:
1. Распаковать содержимое архива wibi.
2. Изменить настройки базы данных в файле configdb.php
3. Перенести все файлы wibi на удаённую машину.
4. В браузере выполнить переход к http://домен/install.php
5. Заполнить все поля: без ошибок.
6. При успешном стечении обстоятельств скрипт установки отредиректит на
главную страницу созданного блога.
7. Удалить install.php.

Использование блога.
Вход в систему осуществляется через ссылку "log in" в правом верхнем
углу страницы. В поле "login" введите "admin", а в поле "password" -
пароль, указанный при инсталяции.
Для управления учётной записью и настройками скрипта перейдите по ссылке
"Admin" в блоке авторизации системы (правый верхний угол).
Для создания новой страницы используйте появившееся при успешной
авторизации поле. Введите в него сообщение. Поле будет увеличиваться по
мере ввода текста. Далее нажмите "create", откроется расширенная форма
создания страницы. Заполните заголовок: он сверху.
Редактирование страниц - по ссылке "edit". При переходе по ней также
имеется возможность удалить сообщение - ссылка "x".
